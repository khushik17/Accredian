# 🚀 Accredian Enterprise Landing Page

A polished and responsive enterprise landing page built using Next.js App Router, focused on clean architecture, smooth user experience, and interactive UI elements.

---

## ✨ Key Features

- Fully responsive design across mobile and desktop  
- Sticky navigation with smooth scrolling between sections  
- Dynamic hero section with animated number counters  
- Interactive **pie chart (mobile)** and **timeline layout (desktop)** for adaptive data visualization  
- Skeleton loaders for smoother perceived performance  
- Dark/Light mode with persistent theme handling  
- Floating **“Chat with Us”** widget powered by API  
- Lead capture form with backend integration  
- Dashboard page to view submitted leads  
- Clean section-based modular architecture for scalability  

---

## 🧩 Tech Stack

- Next.js (App Router)  
- React (Hooks + Functional Components)  
- Tailwind CSS  
- API Routes (backend inside Next.js)  
- Groq API (for chat functionality)  
- Local JSON storage (development)
